1 1. 프로젝트 개요

  • 목표: 팬리스(Fanless) 시스템 구축
  • 중점 요소:
    • CPU, PSU, 저장 장치에서 소음 발생 최소화
    • 공기 흐름이 원활한 매쉬형 케이스 선택
    • 140mm 팬을 저RPM으로 설정하여 발열 문제 해결

2 2. 개별 부품 비교 및 선택 이유

2.1 2.1 CPU 선택

  • 비교 모델: Intel Core i3-13100T vs Intel Core i5-9500
  • 최종 선택: Intel Core i3-13100T (35W 저전력)
  • 이유:
    • 저전력(35W)으로 발열이 적음
    • 4코어 8스레드로 팬리스 환경에서도 성능 유지 가능
    • 최신 DDR5 및 PCIe 5.0 지원 → 확장성 높음

2.2 2.2 CPU 쿨러 선택

  • 비교 모델: NOFAN 아이스파이프 vs Noctua NH-P1
  • 최종 선택: NOFAN 아이스파이프
  • 이유:
    • 팬 없이도 80W 이하 CPU 냉각 가능
    • 크기가 작아 케이스 내부 공간 활용성이 높음
    • 저소음 시스템에 최적

2.3 2.3 마더보드 선택

  • 비교 모델: ASRock B760M-HDV/M.2 (M-ATX) vs ATX 보드
  • 최종 선택: ASRock B760M-HDV/M.2 (M-ATX)
  • 이유:
    • PCIe 5.0 및 최신 DDR5 지원
    • M-ATX 폼팩터로 공간 절약 가능
    • 팬리스 시스템에서 추가 확장이 필요하지 않음

2.4 2.4 스토리지 선택

  • 비교 모델: PCIe 3.0 vs PCIe 4.0 NVMe SSD
  • 최종 선택: Crucial P3 1TB PCIe 3.0 NVMe SSD
  • 이유:
    • PCIe 4.0 대비 발열이 적음
    • 무소음 유지 가능 (HDD 사용 X)
    • 충분한 속도 제공 (3500MB/s)

2.5 2.5 케이스 선택

  • 비교 모델: JONSBO UMX3 vs Fractal Meshify 2 XL vs 3RSYS L600 Quiet
  • 최종 선택: 3RSYS L600 Quiet (미들타워, 140mm 팬 4개 포함)
  • 이유:
    • 기본적으로 저소음 팬 제공
    • 메쉬형 설계로 공기 흐름 극대화
    • 강화유리 패널로 디자인 유지 가능

2.6 2.6 파워 서플라이(PSU) 선택

  • 비교 모델: Seasonic Prime Fanless 500W vs 일반 PSU
  • 최종 선택: Seasonic Prime Fanless 500W (Platinum 인증)
  • 이유:
    • 완전 팬리스 설계로 무소음 유지
    • 고효율 (80+ Platinum)로 발열 최소화
    • 장기간 안정적인 전력 공급 가능

2.7 2.7 냉각 솔루션 최적화 (팬 설정)

  • 비교: 120mm 팬 vs 140mm 팬
  • 최종 선택: 140mm 팬 사용 (기본 장착, 저RPM 설정)
  • 이유:
    • 600~800RPM 저속 설정으로 무소음 유지 가능
    • 120mm 팬 대비 저RPM에서도 동일한 풍량 제공
    • BIOS에서 PWM 모드 설정하여 자동 속도 조절 가능

3 3. 최종 조합 (완전 무소음 PC 빌드)

부품 종류 모델명 특징
CPU Intel Core i3-13100T 35W 저전력, 팬리스 가능
쿨러 NOFAN 아이스파이프 무소음 냉각, 자연 대류 방식
마더보드 ASRock B760M-HDV/M.2 M-ATX, DDR5 및 PCIe 5.0 지원
메모리 삼성 DDR5 16GB x2 (32GB) 듀얼 채널 구성
스토리지 Crucial P3 1TB PCIe 3.0 NVMe 발열 적고 빠른 속도
케이스 3RSYS L600 Quiet 140mm 팬 4개 포함, 저소음 설계
파워 서플라이 Seasonic Prime Fanless 500W 팬리스 PSU, 완전 무소음
팬 설정 140mm 팬 4개 600~800RPM 제한, BIOS에서 속도 조절

4 4. 최종 결론

✅ 완전 무소음 PC 구축 성공 🎯 - CPU, PSU, 스토리지에서 소음 0dB 달성 - 팬리스 냉각 + 140mm 저소음 팬 최적 조합 - BIOS에서 팬 속도 조절하여 필요할 때만 작동 - 매쉬형 케이스로 자연 공기 흐름 확보

🚀 결론: 완전 무소음 + 안정적인 냉각이 가능한 최적의 시스템 구성! 😊✅
